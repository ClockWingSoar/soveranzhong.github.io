# bash 快捷键汇总
## 概述
在 Bash 中，快捷键可以显著提高命令行操作的效率。以下是常用的 Bash 快捷键分类汇总：

---

### **1. 光标移动**
| 快捷键            | 作用                          |
|-------------------|-----------------------------|
| `Ctrl + A`        | 移动到行首                    |
| `Ctrl + E`        | 移动到行尾                    |
| `Alt + B`         | 向后移动一个单词（Backward）  |
| `Alt + F`         | 向前移动一个单词（Forward）   |
| `Ctrl + B`        | 向左移动一个字符（同左箭头）  |
| `Ctrl + F`        | 向右移动一个字符（同右箭头）  |

---

### **2. 编辑操作**
| 快捷键            | 作用                          |
|-------------------|-----------------------------|
| `Ctrl + U`        | 删除光标到行首的内容          |
| `Ctrl + K`        | 删除光标到行尾的内容          |
| `Ctrl + W`        | 删除光标前的一个单词          |
| `Alt + D`         | 删除光标后的一个单词          |
| `Ctrl + Y`        | 粘贴最近删除的内容            |
| `Ctrl + _`        | 撤销上一次编辑                |

---

### **3. 历史命令操作**
| 快捷键            | 作用                          |
|-------------------|-----------------------------|
| `Ctrl + P`        | 上一条命令（同上箭头）        |
| `Ctrl + N`        | 下一条命令（同下箭头）        |
| `Ctrl + R`        | 反向搜索历史命令              |
| `Ctrl + G`        | 退出历史搜索模式              |
| `!!`              | 执行上一条命令                |
| `!$`              | 引用上一条命令的最后一个参数  |

---

### **4. 进程控制**
| 快捷键            | 作用                          |
|-------------------|-----------------------------|
| `Ctrl + C`        | 终止当前运行的命令            |
| `Ctrl + Z`        | 暂停当前进程（放入后台）      |
| `Ctrl + D`        | 退出当前 Shell 或输入结束符   |
| `Ctrl + L`        | 清屏（同 `clear` 命令）       |

---

### **5. 标签页与窗口**
| 快捷键            | 作用                          |
|-------------------|-----------------------------|
| `Ctrl + Shift + T`| 打开新标签页（终端支持时）    |
| `Ctrl + PageUp`   | 切换到上一个标签页            |
| `Ctrl + PageDown` | 切换到下一个标签页            |

---

### **6. 其他实用快捷键**
| 快捷键            | 作用                          |
|-------------------|-----------------------------|
| `Ctrl + X + E`    | 用默认编辑器编辑当前命令       |
| `Alt + .`         | 插入上一条命令的最后一个参数  |
| `Ctrl + XX`       | 在行首和光标位置之间切换      |

---

### **7. 自定义快捷键**
可以通过修改 `~/.inputrc` 文件自定义快捷键。例如：
```bash
# 将 Alt + L 转换为小写当前单词
"\el": "lw"
```

---
## 调用历史命令

在 Bash 中，除了 `!$`（调用上一条命令的最后一个参数）之外，还有其他方式可以调用历史命令的参数。以下是更全面的汇总：

---

### **1. 调用历史命令的参数**
| 快捷键/符号       | 作用                                                                 |
|------------------|---------------------------------------------------------------------|
| `!^`             | 调用上一条命令的第一个参数（等价于 `!:1`）。                         |
| `!$`             | 调用上一条命令的最后一个参数（等价于 `!:$`）。                       |
| `!:n`            | 调用上一条命令的第 `n` 个参数（`n` 从 1 开始计数）。                 |
| `!*`             | 调用上一条命令的所有参数（不包括命令本身）。                         |
| `!:m-n`          | 调用上一条命令的第 `m` 到第 `n` 个参数（范围包含 `m` 和 `n`）。      |
| `!:2*`           | 调用上一条命令的第 2 个参数到最后一个参数。                         |
| `!:2-$`          | 同上，调用第 2 个参数到最后一个参数。                               |

#### **示例**
假设上一条命令是：
```bash
cp file1.txt file2.txt /backup/
```
- `!^` → `file1.txt`
- `!$` → `/backup/`
- `!:2` → `file2.txt`
- `!*` → `file1.txt file2.txt /backup/`
- `!:1-2` → `file1.txt file2.txt`

---

### **2. 调用历史命令的路径**
| 快捷键/符号       | 作用                                                                 |
|------------------|---------------------------------------------------------------------|
| `Alt + .`        | 插入上一条命令的最后一个参数（交互式操作，可多次按 `Alt + .` 回溯）。 |
| `Alt + 0` 到 `Alt + 9` | 插入上一条命令的第 `n` 个参数（`n` 对应数字键）。                   |

#### **示例**
- 输入 `ls -l` 后，按 `Alt + .` 会插入 `-l`。
- 输入 `cp file1.txt file2.txt` 后，按 `Alt + 1` 插入 `file1.txt`。

---

### **3. 调用历史命令的修改**
| 快捷键/符号       | 作用                                                                 |
|------------------|---------------------------------------------------------------------|
| `!!:gs/old/new/` | 替换上一条命令中所有 `old` 为 `new`（全局替换）。                    |
| `!!:s/old/new/`  | 替换上一条命令中第一个 `old` 为 `new`。                              |

#### **示例**
假设上一条命令是：
```bash
echo "hello world"
```
- `!!:s/hello/hi/` → `echo "hi world"`
- `!!:gs/ /_/` → `echo_"hello_world"`

---

### **4. 调用历史命令的快速编辑**
| 快捷键/符号       | 作用                                                                 |
|------------------|---------------------------------------------------------------------|
| `Ctrl + X + E`   | 用默认编辑器打开上一条命令进行编辑（保存后自动执行）。               |

---

### **5. 调用历史命令的路径补全**
| 快捷键/符号       | 作用                                                                 |
|------------------|---------------------------------------------------------------------|
| `Esc + *`        | 将当前命令的参数补全为匹配的文件列表（需启用通配符）。               |

#### **示例**
输入 `ls *.txt` 后按 `Esc + *`，会展开为：
```bash
ls file1.txt file2.txt
```
